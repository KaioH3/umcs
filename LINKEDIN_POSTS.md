## POST 1: Apresentação do Projeto

---

**Acabei de criar um sistema de análise linguística que detecta sarcasmo, discurso de ódio e viés em 15+ idiomas — e roda local, sem API externa, sem custos! 🚀**

O projeto se chama **UMCS (Universal Morpheme Coordinate System)** e transformou a análise de sentimentos.

**O que ele faz:**
- ✅ Detecção de sarcasmo (haha, kkk, loooove, "great", etc)
- ✅ Detecção de discurso de ódio (6 categorias)
- ✅ Análise de sentimento multi-língua
- ✅ Detecção de viés político
- ✅ 70+ idiomas suportados
- ✅ 4.3M palavras no léxico
- ✅ API REST rápidas
- ✅ Zero dependências externas

**Por que isso importa:**
- LLMs têm dificuldade com sarcasmo (literalmente interpretam texto)
- Análise de odio funciona offline
- Sem custos de API
- Código aberto (MIT)

O mais legal? Tudo baseado em raízes etimológicas — "terrible" (EN), "terrível" (PT), "terrible" (ES) = mesma raiz "terr" (Latin terror).

## POST 2: Detecção de Sarcasmo

---

**Sarcasmo é um dos problemas mais difíciles em NLP. LLMs falham feio com isso.**

Mas e se pudéssemos detectar sarcasmo usando apenas padrões linguísticos?

Fiz um sistema que detecta sarcasmo em 15+ idiomas com **zero机器学习**:

**Padrões detectados:**
- 😆 Risadas: haha, lol, kkk, rsrs, jaja, mdr, хаха
- 🔥 Elongação: loooove, soooo
- 💬 Aspas: "great"
- ❓ Perguntas: né?, right?
- 🎭 Contraste: "love this shit"
- 📉 Understatement: just a little

**Exemplos funcionando:**
- "yeah right this is great" → 80% sarcástico
- "kkk que legal" → 50% sarcástico  
- "i loooove this" → 30% sarcástico
- "haha 'great' job" → 90% sarcástico

**Português, Inglês, Espanhol, Francês, Alemão, Russo, Japonês, Coreano... tudo suportado!**

The best part: runs completely offline. Zero API calls.

## POST 3: Discurso de Ódio

---

**Criei um detector de discurso de ódio que roda offline, sem custos de API, em múltiplos idiomas.**

O problema: moderadores de conteúdo gastam fortunas com APIs de IA que custam centavos por análise.

A solução: UMCS detecta 7 categorias de discurso de ódio:

1. **RACISM** - insultos raciais
2. **SEXISM** - machismo
3. **HOMOPHOBIA** - LGBTfobia
4. **RELIGIOUS_HATRED** - ódio religioso
5. **VIOLENCE** - ameaças violentas
6. **ABLEISM/CAPACITISM** - capacitismo
7. **GORDOPHOBIA** - gordobia

**Exemplos de detecção:**
- "you faggot" → HOMOPHOBIA (HIGH)
- "kill all jews" → VIOLENCE (CRITICAL)
- "você é um viado" → ABLEISM (MEDIUM)
- "você é um gordo" → GORDOPHOBIA

**Supported languages:** English, Portuguese, Spanish + more

Zero API calls. Zero costs. Runs locally.

This could be huge for content moderation, right?
